# OpenCV-Mp4-HSV
This project contains a basic python script to run opencv-python with an HSV mask applied to a webcam stream.

## Previews
![Preview1](./preview/preview1.PNG?raw=true "Preview1")

## How it work's

The HSV color space components:
  H – Hue ( Dominant Wavelength ).
  S – Saturation ( Purity / shades of the color ).
  V – Value ( Intensity ).

In the HSV color space only the Hue (H) channel describes color.

## How to setup
- Install opencv-python
```
$ pip install opencv-python
```

- Install tkinter
```
$ pip install tkinter
```

- Install pillow
```
$ pip install pillow
```

- Install numpy
```
$ pip install numpy
```

## How to run
- Run
```
$ python ./app.py
```
